# Тестовое задание для Avito

Полное описание по [ссылке](https://github.com/avito-tech/internship_frontend_2022)

## Технологии

### Frontend

- TypeScript
- React
- React Router
- Redux
- Redux Toolkit
- RTKQuery
- CSS Modules

### Backend

- Node.js
- TypeScript
- Express.js

## Установка

1. Клонируем репозиторий:

```sh
git clone git@github.com:Paavveel/hacker-news.git
```

2. Переходим в папку с проектом:

```sh
cd hacker-news
```

### Backend

3. Переходим в директорию с бэкендом:

```sh
cd server
```

4. Устанавливаем зависимости:

```sh
yarn install
```

5. Запускаем проект:

```sh
yarn start
```

### Frontend

6. Переходим в директорию с фронтендом:

```sh
cd client
```

7. Устанавливаем зависимости:

```sh
yarn install
```

8. Запускаем проект:

```sh
yarn start
```
